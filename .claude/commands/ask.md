# Ask Mode - 질문하기

사용자가 `/ask` 뒤에 질문을 입력했습니다. $ARGUMENTS 를 질문으로 처리하세요.

## 질문 처리 절차

1. 질문 키워드를 분석하여 관련 노트 검색
2. 검색 우선순위:
   - `02_Concepts/` 개념 노트에서 키워드 매칭
   - `01_Weekly/` 강의 노트에서 내용 검색
   - `04_Missions/` 미션 노트에서 관련 내용
   - `05_Resources/` 참고자료에서 보충 정보

3. 찾은 노트를 기반으로 질문에 답변
4. 답변 출처를 명시: "이 내용은 [노트명]에서 확인할 수 있습니다"
5. 관련 개념 노트가 있으면 추가 탐색 제안

## 답변 형식

- 핵심 답변을 먼저 (2~3문장)
- 상세 설명 (노트 내용 인용)
- 관련 개념 링크
- 후속 질문 유도: "더 알고 싶은 부분이 있으신가요?"

## 질문 유형별 처리

- **개념 질문** ("RAG가 뭐야?"): 개념 노트 기반 답변
- **비교 질문** ("RAG vs Agentic RAG 차이?"): 양쪽 노트 참조
- **실습 질문** ("LangGraph 어떻게 써?"): 강의 노트 + 프로젝트 노트 참조
- **미션 질문** ("Week05 미션 힌트"): 미션 노트 참조
