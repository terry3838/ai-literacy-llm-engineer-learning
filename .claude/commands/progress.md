# Progress - 학습 진도 확인

사용자가 `/progress` 또는 `진도`를 입력했습니다. 아래 절차대로 진행하세요.

## 진도 확인 절차

1. `.claude/learning-progress.json` 파일 읽기
2. 파일이 없으면: "아직 학습 기록이 없습니다. `/learn`으로 학습을 시작해보세요!" 안내

## 표시 내용

### 1. 전체 진행률
```
전체 진행률: [완료 Day 수] / [전체 Day 수] (XX%)
현재 위치: Week[X] Day[Y]
```

### 2. 주차별 진행 상황
```
Week 01 AI Literacy          [===---] 60%
Week 02 자료구조/알고리즘      [======] 100%
Week 03 Git/Docker            [====--] 80%
Week 04 네트워크/클라우드      [------] 0%
...
```

### 3. 퀴즈 성적 (있는 경우)
```
최근 퀴즈 결과:
  Week05 (보통) - 8/10 (80%)
  Week02 (쉬움) - 10/10 (100%)

약한 영역: topic/rag, topic/prompt-engineering
```

### 4. 추천 다음 단계
- 학습하지 않은 주차가 있으면 다음 주차 학습 추천
- 퀴즈 점수가 낮은 영역이 있으면 해당 복습 추천
- 미션을 안 한 주차가 있으면 미션 도전 추천

## 후속 안내

- "학습을 이어하시겠습니까?" -> `/learn`
- "약한 부분을 복습하시겠습니까?" -> `/review`
- "미션에 도전하시겠습니까?" -> `/mission`
