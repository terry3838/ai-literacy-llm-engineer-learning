# Gradio

> [!abstract] 한줄 정의
> Python 기반의 ML 데모 및 인터랙티브 UI 프레임워크.

## 핵심 이해

Gradio는 머신러닝 모델의 데모 UI를 몇 줄의 코드로 구축할 수 있는 프레임워크다. 챗봇 인터페이스, 파일 업로드, 스트리밍 출력 등을 쉽게 구현할 수 있어 LLM 기반 에이전트의 프론트엔드로 활용된다.
