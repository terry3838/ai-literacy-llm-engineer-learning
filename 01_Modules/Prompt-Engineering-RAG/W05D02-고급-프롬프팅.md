# 고급 LLM 프롬프팅 전략

## 핵심 개념

> [!summary] 요약
> 기본 프롬프팅을 넘어 고급 전략을 학습한다. Chain-of-Thought(CoT) 추론, Few-shot/Zero-shot 학습, 출력 형식 제어, 자기 일관성(Self-Consistency), Tree-of-Thought 등 다양한 고급 프롬프팅 기법을 통해 LLM의 추론 능력을 극대화하는 방법을 다룬다.

## 주요 내용

### 1. Chain-of-Thought (CoT) Prompting
- 모델에게 **단계별 추론 과정**을 유도하는 기법
- "Let's think step by step" 등의 지시어 활용
- 복잡한 문제 해결 시 정확도 향상
- 관련: [[Chain-of-Thought]]

### 2. Few-shot / Zero-shot Learning
- **Zero-shot**: 예시 없이 지시만으로 수행
- **Few-shot**: 소수의 예시를 제공하여 패턴 학습 유도
- 예시의 품질과 다양성이 중요
- 관련: [[Few-shot Learning]], [[In-context Learning]]

### 3. 출력 형식 제어
- JSON, 마크다운, 표 등 **구조화된 출력** 유도
- 출력 스키마 명시적 정의
- 관련: [[프롬프트 구조]]

### 4. Self-Consistency
- 동일 질문에 여러 번 추론 후 **다수결** 방식으로 최종 답 선택
- CoT와 결합하여 정확도 향상
- 관련: [[Self-Consistency]]

### 5. Tree-of-Thought (ToT)
- 하나의 추론 경로가 아닌 **여러 사고 경로를 탐색**
- 각 경로를 평가하고 가장 유망한 경로 선택
- 복잡한 의사결정 문제에 효과적
- 관련: [[Tree-of-Thought]]

> [!tip] 실전 팁
> 프롬프팅 기법은 단독으로 사용하기보다 **상황에 맞게 조합**하는 것이 효과적이다. 간단한 작업은 Zero-shot, 복잡한 추론은 CoT + Few-shot 조합을 고려한다.
