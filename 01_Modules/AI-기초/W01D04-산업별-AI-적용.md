# 산업별 AI 적용 사례 탐구

## 핵심 개념

> [!summary] 요약
> AI 적용을 생활/문화, 기술/경제, 사회/공공 3축으로 정리한다. 교육은 맞춤형 학습과 교사 업무 자동화, 의료는 신약개발과 진단보조(HITL), 제조는 자율제조와 디지털 트윈, 금융/물류/공공까지 전 산업에 걸친 AI 확산 사례를 다룬다.

## 주요 내용

### 1. 생활/문화 -- 교육 AI

- **맞춤형 학습**: AI 튜터를 통한 개인화 자료 추천, 학습 경로 분석
- **교사 조력**: 업무 자동화, 교수법/평가 지원
- **접근성 향상**: 언어/형식 장벽 완화, 교육 소외계층 격차 완화
- **5대 과제**: 정확성 / 안전성(프라이버시) / 비판적 사고 / 부정행위 / 형평성
- **사례**: 학습 전용 LLM -- 소크라테스식 질문으로 유도하며 바로 정답 대신 추론을 유도하는 질문 설계

### 2. 생활/문화 -- 의료/웰니스 AI

- **신약 개발**: 단백질 3D 구조 예측 -> 연구/신약개발 가속
- **의료 어시스턴트**: 의사 발화 -> 전자의무기록(EMR) 자동 생성 (시간 절감)
- **진단/진단보조(HITL)**: AI 경고/예측 + **사람이 최종 판단**
  - 예: 심전도 분석, 폐암 재발 가능성 예측(임상기록+검사기록 종합)
- **웰니스/스포츠**: 동작 인식(비전) + LLM 코칭 결합 -> 자세/방향 교정

> [!tip] 의료 AI 안전 설계 원칙
> 1. AI 역할을 "결정"이 아니라 **조력(추천/경고/요약)**으로 제한
> 2. 임상의 검토 지점(HITL)을 **필수 게이트**로 설계
> 3. 오류 시 피해를 줄이는 **안전장치(알림, 로그, 근거 제시)** 탑재
> 4. 데이터/프라이버시/규제 준수

### 3. 생활/문화 -- 일상 어시스턴트/가정 로봇

- **일상 어시스턴트**: 스케줄 관리, 스마트홈 제어, 여행 계획 보조
- **가정 로봇**: 감정 인식 + 대화, 1인 가구/노인 케어의 "대화 상대/생활 보조" 가능성
- **로봇 핵심 기술**: SLAM 등 위치추정/지도화 (후반 물류/자율주행 파트로 이어짐)

### 4. 생활/문화 -- 생성형 창작/게임/3D/전시

- **생성형 창작**: 텍스트 -> 이미지/영상 도구 확산
- **게임**: AI 팀메이트/코플레이어로 상호작용 강화
- **3D**: 3D 생성 도구로 모델링 접근성 상승
- **전시/도슨트**: 관람객 맞춤형 설명, 인터랙션 기반 안내

### 5. 기술/경제 -- 제조/자율제조/디지털 트윈

- **자율제조**: AI + 디지털 트윈 + 로봇의 결합으로 "자율제조(인간 개입 최소)"를 지향
- **디지털 트윈**: 현실을 가상 공간에 복제 -> 시뮬레이션 -> 최적 방안 도출 -> 현장 적용
- **품질/안전**: AI가 실시간 품질 판단 -> 이상 시 작업자 알림

### 6. 기술/경제 -- 플랫폼/커머스/유통/물류

- **광고/커머스**: 도달/클릭률 분석 -> 캠페인 최적화/자동화, 사용자 맥락 기반 추천 고도화
- **SCM/물류 예측**: 재고 관리, 주문 예측, 배치/출고 최적화
- **운송 최적화**: 실시간 교통/날씨/주문량 고려 -> 연료/시간 최소화
- **자율주행 배송/SLAM**: 위치 인식 + 지도 생성 기반으로 이동/배송
- **로봇 물류 센터**: AMR(자율주행) vs AGV(가이드/레일 기반)

### 7. 기술/경제 -- 금융/IT/건설

- **금융**: AI 기반 투자 조언, 고객 응대 자동화(언어 장벽 완화), AI 신용평가
- **IT/반도체**: GPU 대체 저전력 고속 처리 장치, BCI(뇌-컴퓨터 인터페이스) 초기 단계
- **건설**: BIM + AI로 설계 자동화/비용 예측, 비전/음향 센서로 안전 감지, 드론 기반 자율주행 건설 로봇

### 8. 사회/공공 -- 행정/스마트시티/교통/국방/에너지/농업

- **행정**: 민원 자동 분류/응답, 정책 시뮬레이션
- **스마트시티**: 교통 흐름 최적화, 에너지 관리
- **국방**: 자율 무기 시스템, 정보 분석
- **에너지/환경/농업**: 전력 수요 예측, 작물 모니터링
- **우려**: 프라이버시, 자율무기, 전력/물 소비 문제
