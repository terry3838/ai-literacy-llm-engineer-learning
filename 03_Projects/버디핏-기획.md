## 서비스 개요

> [!summary] 한줄 설명
> AI 기반 운동 습관 형성 플랫폼 - 개인 맞춤 운동 추천과 버디 시스템으로 지속적인 동기 부여

## 서비스 구조

```mermaid
flowchart TD
    User[사용자] --> App[버디핏 앱]
    App --> Agent[AI 운동 버디\nAgent]
    Agent --> Recommend[운동 추천]
    Agent --> Track[습관 추적]
    Agent --> Motivate[동기 부여]

    Agent --> LLM[LLM\n대화/분석]
    Agent --> DB[(사용자 데이터)]
```

## 핵심 기능

### 1. AI 운동 추천
- 사용자 프로필 기반 맞춤 운동 제안
- 난이도 자동 조절

### 2. 버디 시스템
- AI 버디와의 대화형 인터랙션
- 응원, 피드백, 습관 리포트

### 3. Agentic Workflow
- 사용자 데이터 분석 → 운동 계획 생성 → 실행 모니터링 → 피드백
